# red-blue-nim-gui game

This project implements the **Red-Blue Nim game** with a **Tkinter-based graphical user interface** and an **AI player** using optimal game strategy.

## Features
- Two piles: Red and Blue marbles
- Players can remove 1 or 2 marbles from a single pile per turn
- Game variants:
- **Standard**: Player who empties a pile wins
- **Misere** : Player who empties a pile loses
- Modes:
- Human Vs Human
- Human Vs AI
- AI vs AI
- Optimal AI using dynamic proggramming
- Automatic gameplay option
- Generates:
- Markdown report
- PDF report
- Graphs of game progression

## Requirements
-Python
-Tkinter(included with standard Python)
matplotlib
